Webbens historia

En resa genom webbens utveckling från 1990-talet till dagens moderna internet

Från statiska sidor till interaktiva plattformar

Webben har genomgått en revolutionerande utveckling sedan Tim Berners-Lee skapade World Wide Web 1990. Från enkla textdokument till dagens kraftfulla webbapplikationer - varje era har fört med sig nya möjligheter och förändrat hur vi arbetar, kommunicerar och lever.

Tidslinje över webbens utveckling

1990-1999

Webbens födelse

Tim Berners-Lee skapar World Wide Web vid CERN. De första webbsidorna innehåller endast text och enkla länkar. HTML, HTTP och URL:er blir grundstenarna för det moderna internet.

Detaljerad tidslinje →
1995-2000

Web 1.0 - Den statiska webben

Webbsidor är statiska dokument med begränsad interaktivitet. Netscape Navigator och Internet Explorer konkurrerar om marknadsandelar. JavaScript introduceras för att skapa mer dynamiska upplevelser.

Utforska Web 1.0 →
2004-2010

Web 2.0 - Den sociala revolutionen

Användare blir innehållsskapare. Facebook, YouTube och Wikipedia förändrar hur vi delar information. AJAX gör webbsidor mer interaktiva och responsiva utan att behöva ladda om hela sidan.

Upptäck Web 2.0 →
2007-2015

Mobilrevolutionen

iPhone lanseras 2007 och förändrar hur vi använder internet. Responsiv webbdesign blir avgörande. Pekskärmar och appar skapar nya designmönster och användarförväntningar.

Mobil & PWA →
2015-Idag

Modern webb & PWA:er

Progressive Web Apps (PWA:er) suddar ut gränserna mellan webb och appar. AI, WebAssembly och avancerade Web API:er öppnar för nya möjligheter. Webben blir en kraftfull applikationsplattform.
